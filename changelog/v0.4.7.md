# v0.4.7 — Global Quick Search & Recent Tables

**Release Date:** March 2026

Focused UX and navigation release adding a global quick-search palette for fast table access in the active connection, plus recent-table history and supporting performance cleanup.

## Completed Tasks

### Quick Search

- [x] Add global quick search palette opened by `Cmd/Ctrl+K`
- [x] Scope results to the active sidebar connection
- [x] Support keyboard-first navigation with `Up`, `Down`, `Enter`, and `Escape`
- [x] Open selected tables directly in the existing data-tab flow
- [x] Add top-bar quick-search trigger next to settings

### Recent Tables

- [x] Persist lightweight recent-table references in app state
- [x] Update recent-table history from the canonical `openTableTab(...)` path
- [x] Show a `Recent tables` section when the quick-search query is empty
- [x] Scope recents by `connectionId` and ignore stale references that no longer resolve

### Performance & Structure

- [x] Move quick-search ranking and matching into a pure helper module
- [x] Add deferred query filtering so typing stays responsive
- [x] Cap palette rendering to a bounded result set to avoid excessive DOM work
- [x] Add focused Bun coverage for normalization, ranking, schema matching, recent scoping, and limits
